## Prerequisites

1. **Baseline sweep must be complete**: `mil/mil_results/seg_mil/all_configs.json` must exist with 16 entries (4 frontends × 4 aggregators). If missing, run feature 004 first.

2. **Embedding cache must exist**: `mil/seg_embedding_cache/` must be populated. This was generated by feature 004's `--precompute-only` step.

3. **Seen-child split CSVs** must be at `whisper-modeling/seen_child_splits/`.

4. **Environment**: Activate `child-vocalizations` conda env (Python 3.11):
   ```bash
   conda activate child-vocalizations
   ```

## Troubleshooting

| Symptom | Cause | Fix |
|---------|-------|-----|
| `KeyError: noisy_or` in build_aggregator | seg_model.py not updated | Check that `NoisyORAgg` is registered in `build_aggregator()` |
| NaN loss with `transformer` aggregator | Positional encoding out-of-bounds | Ensure max_len in TransformerAgg ≥ K_max from config |
| Empty `test_segment_weights.csv` for non-attention configs | Expected — only attention/gated_attention/transformer produce weights | Weak diarization eval skips non-attention configs automatically |
| Age-band AUROC missing in `all_configs.json` | Old entries from 16-config baseline | Re-run inference-only step or manually backfill with `--age-band-only` flag |
| `weak_diarization_eval.csv` has NaN correlations | Clips with no ground-truth child segments | Script skips clips with zero GT child coverage; NaN means entire subset had no child speech |
