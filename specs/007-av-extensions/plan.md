# Implementation Plan: AV Extended Experiments — 007-av-extensions

**Branch**: `007-av-extensions` | **Date**: 2026-04-24 | **Spec**: `specs/007-av-extensions/spec.md`

## Summary

Extends the 006 audio-visual fusion pipeline with: (1) a three-stage cascaded detection pipeline reusing existing BabAR/VTC RTTM outputs and ECAPA enrollment scores; (2) temporal smoothing (Gaussian/majority-vote/moving-average) as a post-processing layer over any model's predictions; (3) GPT-4o-mini vision API feature extraction over sampled video frames; (4) LocoNet and Light-ASD as additional ASD frontends alongside TalkNet; (5) optional Ego4D reference experiment; (6) optional 1kd dataset compatibility check. No new model architecture training is required for stages 1–2; all new components are threshold-tuned or post-processed on the validation set.

## Technical Context

**Language/Version**: Python 3.11 (conda `child-vocalizations` env — same as existing AV pipeline)
**Primary Dependencies**: scikit-learn (threshold tuning, logistic regression), pandas, numpy, scipy (Gaussian smoothing), OpenCV (frame sampling), openai (GPT-4o API), tqdm; LocoNet and Light-ASD require the `video/` Python 3.10 uv env as subprocess targets
**Storage**: CSV files for features and predictions; JSON for metrics/configs/cache; `.pkl` for trained models; `av_fusion/gpt4o_cache/` for raw API responses; `av_fusion/face_track_cache/` shared with 006
**Testing**: Manual acceptance tests per user story (run script → check output file schema and metrics)
**Target Platform**: Linux (SLURM cluster), single GPU for GPT-4o frame extraction (CPU-bound; GPU for LocoNet/Light-ASD inference)
**Project Type**: CLI pipeline scripts
**Performance Goals**: Full GPT-4o extraction ≤ $2 total cost; temporal smoothing < 60s for 2183 clips; cascade pipeline < 5 min for full test set
**Constraints**: Test set never used for threshold tuning; Ego4D and 1kd are optional/conditional; GPT-4o cost must be logged before any API calls are made; all new artifacts stored under `av_fusion/av_results/{run_name}/`
**Scale/Scope**: 2183 clips (1311 train / 431 val / 441 test); SAILS BIDS dataset only for video features; seen-child split from `whisper-modeling/seen_child_splits/`
