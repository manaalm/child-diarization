# `encoders/` — frozen-backbone audio encoder baselines

Relocated from `baselines/` to its own top-level module per spec 022 US4 (2026-05-12, PI thesis revisions). The relocation is `git mv` for tracked files and plain `mv` for the untracked run scripts; backwards-compatible import shims live at the old `baselines/` paths for one release cycle.

## Contents

| File | Purpose |
|---|---|
| `baseline_encoders.py` | `EncoderBaseline` class: frozen Whisper / WavLM / Fused backbone + mean or attention pooling + linear classifier head. Three encoders × two poolings = 6 baselines. The Fused variant concatenates Whisper and WavLM embeddings before pooling. |
| `run_fused_attn_unfreeze2_backbone_swap.py` | Backbone-sweep runner: swap Whisper-{base, small, medium, large-v3} under the same fused-attn-unfreeze2 recipe. |
| `run_fused_attn_unfreeze2_kfold.py` | Within-child 3-fold variant of the fused-attn-unfreeze2 setup. (Note: superseded by spec 022 US2 group-stratified k-fold — see `evaluation/group_stratified_kfold.py`.) |
| `yamnet_worker.py` | TFHub YAMNet subprocess worker (spec 022 US3); runs inside `yamnet-eval/.venv/` and emits AudioSet child-vocalisation probabilities to stdout. Not an "encoder baseline" in the classical sense but lives here because it's a frozen audio classifier. |

## Backwards-compatible import shims

For one release cycle (until next user-facing spec), the following stub modules at the OLD `baselines/` paths re-export from `encoders/`:

- `baselines/baseline_encoders.py`  →  `from encoders.baseline_encoders import *`
- `baselines/run_fused_attn_unfreeze2_backbone_swap.py` (run-script shim)
- `baselines/run_fused_attn_unfreeze2_kfold.py` (run-script shim)

Update direct imports/invocations to the new paths at your convenience. The shims will be removed in a future cleanup spec.

## Architecture (matches `docs/figures/encoder_pipeline.png`)

```
input waveform (16 kHz mono)
        │
        ▼
  frozen encoder (Whisper / WavLM / Fused)
        │  per-frame hidden states (T × D)
        ▼
  pooling (mean | attention)
        │  clip embedding (D)
        ▼
  linear classifier → P(child vocalising)
```

The Fused variant runs Whisper and WavLM in parallel, concatenates their per-frame hidden states along the channel axis, then applies pooling and a single linear head.
